# VSCode-Customization

## 🔌 Core Extensions

To achieve this exact look and feel, you need these four extensions:

* **[Apc Customize UI++](https://www.google.com/search?q=https://marketplace.visualstudio.com/items%3FitemName%3Ddtopuzov.vscode-apc-extension)**: Enables the custom gradient cursor and UI scaling.
* **[Fig](https://www.google.com/search?q=https://marketplace.visualstudio.com/items%3FitemName%3Dfig.fig)**: Provides the base theme and enhanced terminal autocomplete.
* **[Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)**: Delivers sleek, minimalist file icons.
* **[VSCode Animations](https://www.google.com/search?q=https://marketplace.visualstudio.com/items%3FitemName%3Dbrandonkirbyson.vscode-animations)**: Powers the smooth UI transitions.

## 🛠️ Quick Start

### 1. The Font

This config uses **Dank Mono**. If you don't have it, replace it in the settings with **JetBrains Mono** or your favorite coding font.

### 2. The Setup

1. Press `Ctrl+Shift+P` and type **"Open User Settings (JSON)"**.
2. Paste the configuration.
3. **Action Required:** Update the `apc.imports` path to your local Windows path (e.g., `C:/Users/YourName/...`) so animations load correctly.

---

## 🕹️ Features

* **Toggle Sidebar**: Press `Ctrl+B`. Because the sidebar is on the **Right**, your code won't "jump" when you open it.
* **Zen Cursor**: Watch the cursor smoothly slide with `editor.cursorSmoothCaretAnimation` enabled.
* **Clean Tabs**: Only the active file shows in the header—no more "tab-overload."
